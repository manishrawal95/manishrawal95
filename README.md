# Hi, I'm Manish 👋

I'm an AI PM building things I actually want to use.

Most of my work is hands-on — real products, not demos. I care about the decision layer: where does the AI make a call, how do you know when it's wrong, and how does it get better from feedback.

---

## What I'm Building

**[RoleRadar](https://roleradar.xyz)** — an AI job search advisor that replaces keyword alerts with an agent that reads JDs, evaluates fit, and learns your preferences over time.

The interesting product problems: hard guardrails before the LLM (accuracy + trust), narrative memory over vector search (reasoning context survives), feedback loop built on revealed preference not intake forms, Today Dashboard as the default surface (job searching needs a daily habit).

Product case study → [github.com/manishrawal95/RoleRadar](https://github.com/manishrawal95/RoleRadar)
